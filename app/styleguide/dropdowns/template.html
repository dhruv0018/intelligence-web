Dropdown Buttons

Dropdown Button Types

Dropdown button types are similar in style to default buttons, but have a different set of behaviors and best practices. Dropdowns open downward to reveal a list of options to a user. The user can then select an option from the list or multiple options if there is a need to do so.

Best Practices:

  • Dropdowns should only be used for items having at least 3 choices.
    • If these options are likely to grow, then use a dropdown from the beginning.
  • Dropdowns involving more than 20 choices should be avoided or at least have some form of sorting or searching.
  • It should be clear to the user when the dropdown is active, when it has been opened, and what choice they are going to make.

Styles

Dropdown buttons follow all the same styles for a default button including sizing, color and stroke, text sizes, and placement. Where they differ is the addition of a small triangle icon to indicate that they open downward when clicked on.

Standard Dropdown

Every dropdown will use the standard style unless it falls into one of the categories below.

  • Pick a Role (Add Role)
  • Season Dropdown (Insights, Analytics)
  • Stat Types
  • Game Types (Stats)

Check Box Dropdown

Dropdown style used when a user can select multiple options at once. Options are selected as a user checks each box.

When more than one option is selected, the dropdown will read "X Selected".

  • Film Home Filters
  • Shot Chart Player Filters
